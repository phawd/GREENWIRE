# Contactless Kernel Notes

This directory contains commented operator notes for the public mapping of:

- `C-2` PayPass
- `C-3` VISA
- `C-4` AMEX
- `C-5` JCB
- `C-6` DISCOVER
- `C-7` CUP

These files are not reproductions of the proprietary Book `C-x` specifications.
They are working notes built from public EMVCo/process material plus public
scheme guidance where available.
