## Requirements

- Python 3.8 or higher

## Installation

Clone the repository and install the package:

```bash
git clone https://github.com/phawd/GREENWIRE.git
cd GREENWIRE
pip install -e .
```

## Usage

Run the main entry point:

```bash
python -m greenwire
```

Or use the installed command-line interface:

```bash
greenwire
```
